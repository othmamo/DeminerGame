# DeminerGame
Minesweeper with c#/.NET using windows forms<br/>

To play, launch the program and choose the difficulties.<br/>

Commands in game:<br/>
  Click on a cell to reveal it.<br/>
  Right click to put a flag on the cell.<br/>
